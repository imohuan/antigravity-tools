# 2026-07-04

## Accounts 页面：导入/导出账户配置 + 表格多选

### 修改的文件
- `web/api/accounts.py`：添加了 `POST /api/accounts/import` 和 `POST /api/accounts/export` 两个端点
- `web/static/index.html`：前端 UI 和 Vue 逻辑

### 功能细节
1. **表格多选**：每行左侧添加复选框，表头有全选复选框
2. **选择状态栏（表格下方）**：选中账户时表格下方显示 "已选中 X / Y 个账户" 及全选/反选/清空快捷按钮
3. **导入对话框**：支持文件上传（`.json`）和粘贴内容两种模式，自动解析 `{accounts: [...]}` 或 JSON 数组格式，预览后批量导入
4. **导出对话框**：展示已选中账户列表，直接操作复制到剪贴板或下载 `.json` 文件。未选中时 toast 提示
5. **导入导出数据格式**：JSON，每个条目包含 method/apikey/access_token/uid/nickname/domain 字段
